Installation Instructions

  Download: Download the file for your OS and the import-assistant-labels.ny script from the Releases tab.
  
  Extract: Right-click the zip file and select Extract All.
  
  Install: Copy the entire folder named EditorAssistant.vst3 into your VST3 directory:

    Windows	C:\Program Files\Common Files\VST3
    macOS	/Library/Audio/Plug-Ins/VST3 (or ~/Library/Audio/Plug-Ins/VST3)
    Linux	~/.vst3 (or /usr/lib/vst3)
  
  Rescan: Open Audacity and go to Effect > Plugin Manager. Click Rescan, find "Editor Assistant," and click Enable.

  The Nyquist script can be installed in Audacity using Tools -> Nyquist Plugin Installer -> import-assistant-labels.ny -> Apply

🛠️ How to Use

    Access: Once enabled, find the plugin under the Effect menu (or the Real-Time Effects sidebar in Audacity 3.2+).

    Processing: Select the audio track you wish to analyze and open the Editor Assistant interface.

    Import: Labels can be  imported with the Nyquist script -> Analyze -> Import Assistant Labels
